## 常用快捷键

| 快捷键 | 符号      |
| ------ | --------- |
| ^      | 指数/上标 |
| /      | 分号      |
| _      | 指数/下标 |
| e      | 自然数e   |
| ⌥+V    | 平方根号  |
| ⇧+⌥+V  | 三次根号  |
| ⌥+W    | 求和号    |
| ⇧+⌥+P  | 求积号    |
| ⌥+B    | 积分      |
| ⌥+P    | 圆周率    |
